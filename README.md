# Backend Engineering Bootcamp 2026

A structured repository documenting my journey toward becoming a professional Backend Engineer.

This repository serves as my engineering notebook throughout the bootcamp. It contains daily exercises, technical notes, learning resources, and progress logs while building a strong foundation in backend development and software engineering.

---

## 🎯 Goals

- Build a solid foundation in Backend Engineering
- Master Python for backend development
- Learn Git and professional development workflows
- Design and build RESTful APIs
- Understand database design and SQL
- Develop production-ready software engineering practices
- Document the learning process consistently

## 📁 Repository Structure

```text
backend-bootcamp-2026/
├── notes/
├── exercises/
├── resources/
├── daily-log/
```

- **notes/** – Technical notes and concept summaries

- **resources/** – Books, articles, videos, and useful references
- **daily-log/** – Daily learning journal and progress tracking
- **templates/** – Reusable project and documentation templates
